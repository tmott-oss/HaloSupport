# Halosight Sales Process — Codex Pack

This pack contains the Halosight sales process structured for AI-agent enforcement.

Source basis: Halosight Sales Process PDF, page 1.

Contents:
- `global/sales_process_stages.md` — stage definitions, qualifications, and exit criteria
- `logic/ai_enforcement_rules.md` — if/then advancement, regression, stall, and red-flag logic
- `logic/validation_dictionary.md` — objective definitions for what counts as qualified evidence
- `logic/deal_score_model.md` — weighted scoring model, score bands, and forecast logic
- `schemas/opportunity_data_model.yaml` — normalized fields for deal inspection
- `examples/deal_inspection_output_examples.md` — example JSON outputs
- `examples/codex_system_prompt.md` — suggested operating prompt for a Halosight sales-process agent

Use this pack as the source of truth for opportunity stage enforcement, qualification discipline, and deal inspection.
