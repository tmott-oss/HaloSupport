# Halosight Codex Context Pack

This directory contains a Codex-ready Markdown context pack for a Halosight Sales AI Agent Team.

## Structure

- `global/` — company truth, product framing, proof boundaries, sales process, qualification logic
- `personas/` — stakeholder-specific battlecards and approved messaging
- `competitive/` — competitor and status-quo positioning
- `stages/` — stage-specific selling guidance
- `accounts/TEMPLATE/` — local account-context templates for deal-specific agents

## Important operating note

This pack is a strong internal working `v1`, not a substitute for verified product, security, or legal source-of-truth files.

Agents using this pack must:

- avoid inventing unsupported claims
- treat unverified product details as needing confirmation
- treat security, compliance, and ROI claims as restricted unless explicitly approved
- avoid public customer references unless they are approved proof assets
