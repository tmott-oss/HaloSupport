# Stage Guidance — Decision

## Goal

Reduce perceived risk and support next-step motion.

## Approved decision-stage framing

Start with the foundational workflow that creates clear user value and measurable operating lift. Phase the rollout so data capture is fixed first, then expand workflows and analytics.

## Decision-stage priorities

- stakeholder alignment
- pricing and packaging review
- security and technical review
- Salesforce design review
- phased implementation plan
- success criteria

## Desired outcome

The buying group agrees on concrete next steps toward purchase.
