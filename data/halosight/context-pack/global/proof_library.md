# Halosight Proof Library

## Status

This file contains approved proof themes and restrictions for AI sales agents. It is not yet a verified external proof source.

## Approved generic proof themes

- less manual note burden
- better capture of commitments and next steps
- stronger CRM data quality
- improved manager visibility
- more structured field intelligence
- phased rollout reduces adoption risk

## Customer story restrictions

Known from context:

- FleetPride is an active account / opportunity context, but should not be presented as a public case study unless approved.
- Challenger Gray appears to be a client/prospect context, but should not be presented as public proof unless approved.

## Approved proof usage

Agents may use directional proof language such as:

- helps reduce rep admin burden
- helps improve data capture consistency
- helps create more inspectable field activity
- helps leadership gain better visibility into what is happening with accounts

## Restricted proof usage

Agents may not use the following unless specifically approved:

- named customer logos
- public case-study claims
- customer quotes
- quantified ROI claims
- benchmark claims vs competitors
