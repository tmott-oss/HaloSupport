# Halosight Truth Boundaries

## Safe external claims

- Halosight helps capture, summarize, and structure information from customer interactions.
- Halosight is designed to reduce manual note burden and improve CRM data quality.
- Halosight supports phased implementation.
- Halosight is especially relevant for field and outside sales teams.
- Halosight can surface commitments, customer signals, deal risks, pricing pressure, and key takeaways from interactions.

## Claims requiring verification before external use

- exact module names
- exact supported workflows
- exact Salesforce write-back behavior
- exact AI Assistant capabilities
- exact integration architecture
- specific security certifications
- exact ROI numbers
- exact customer names / logos / public references
- “real-time” claims unless proven true in production
- “automatically inserts into Salesforce” unless conditions and limits are known

## Hard do-not-say list

- guaranteed ROI claims
- unsupported security/compliance claims
- unsupported competitor parity claims
- public reference to a client without approval
- “replace Salesforce”
- “eliminate managers”
- “works out of the box for every sales org”
- “fully autonomous selling”

## Security and compliance caution

Do not claim any of the following unless verified in an approved source file:

- SOC 2 Type II certified
- HIPAA compliant
- zero data retention
- customer data is never stored
- all data is encrypted end-to-end
- customer data is never used for training
- native support for a specific SSO / IAM provider
- exact deployment architecture

## ROI caution

Approved without numbers:

- can reduce rep admin
- can improve data completeness
- can improve visibility
- can support stronger follow-up discipline

Not approved without validation:

- “improves revenue by X%”
- “cuts admin by Y hours per week”
- “improves forecast accuracy by Z%”
- “increases win rate by X%”
