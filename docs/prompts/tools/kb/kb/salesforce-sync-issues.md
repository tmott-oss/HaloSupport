# Salesforce Sync Issues

## Problem

The user says notes, summaries, or other Halosight data are not showing up in Salesforce.

## What to check

1. Make sure the Salesforce account is connected
2. Make sure the user is logged into the correct Salesforce account
3. Make sure required field mappings are configured
4. Make sure the sync process completed successfully
5. Check whether this is a known issue

## Simple troubleshooting steps

1. Ask the user to reconnect Salesforce
2. Ask the user to confirm field mappings are active
3. Ask the user to try the action again
4. If the issue continues, collect diagnostics and escalate

## Collect these details if unresolved

* user email
* company name
* browser
* device
* exact error message
* whether reconnect was attempted
* whether field mapping was confirmed

## Escalate if

* reconnect does not fix the issue
* the user reports missing data
* the issue appears account-specific
* the agent is not confident

## Important rule

Do not guess why the sync failed.
If basic steps do not work, escalate to human support.
