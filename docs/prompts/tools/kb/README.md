# Knowledge Base

This folder contains the approved support content the Halosight Support Agent is allowed to use.

## Rules

* If the answer is not in this folder, the agent should not guess
* Articles should be simple, clear, and step-by-step
* Each article should focus on one problem

## First topics to add

* Password reset
* Login problems
* Salesforce sync issues
* Missing notes or summaries
* Mobile app upload issues
* Basic setup help
